# Feedly AnyLayout

Chrome расширение, которое позволяет навигировать по статьям в Feedly клавишами `j`/`k` в **любой (русская\английская) раскладке** клавиатуры.

## Проблема

На Feedly для навигации между статьями используются клавиши:
- `j` — следующая статья
- `k` — предыдущая статья

Но если забыть переключить раскладку на английскую, навигация перестаёт работать.

## Решение

Это расширение перехватывает нажатия клавиш и эмулирует английскую раскладку:

| Раскладка | Клавиша | Действие |
|-----------|---------|----------|
| Английская | `j` | Следующая статья |
| Английская | `k` | Предыдущая статья |
| Русская | `о` | Следующая статья |
| Русская | `л` | Предыдущая статья |

## Установка

### Из исходного кода

1. Откройте `chrome://extensions/`
2. Включите **«Режим разработчика»** (справа сверху)
3. Нажмите **«Загрузить распакованное»**
4. Выберите папку с расширением

## Использование

После установки расширение автоматически работает на `feedly.com`. Просто нажимайте `j`/`k` или `о`/`л` для навигации — не нужно думать о раскладке!

## Особенности

- ✅ Работает только на `feedly.com`
- ✅ Игнорирует поля ввода (не мешает печатать)
- ✅ Поддерживает комбинации с модификаторами
- ✅ Не требует прав доступа к данным
